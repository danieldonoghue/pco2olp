# Screenshots

Place screenshot images here. File names referenced in the documentation:

| File | Description |
|---|---|
| `main-window.png` | Main window with a plan loaded and items listed |
| `settings.png` | Settings dialog |
| `generate-progress.png` | Generation progress dialog |
| `generate-done.png` | Post-generation dialog |
| `authentication.png` | Browser OAuth prompt |
| `gatekeeper.png` | macOS Gatekeeper warning and System Settings resolution |
| `smartscreen.png` | Windows SmartScreen warning |

Recommended resolution: 2x (retina/HiDPI) PNG, cropped to the relevant area.
